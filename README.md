## 🛠️ Instalação
1. Clone o repositório:
```bash
git clone https://github.com/diogo-kappaun/noteflix
```
2. Vá até o repositório clonado:
```bash
cd noteflix-web
```
3. Instale as dependências:
```bash
npm install
```
4. Execute o app:
```bash
npm run dev
```
